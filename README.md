# How to play

1. You need to download the following files
> start.bat, main.py & the images Folder

2. Start the start.bat file

3. Now you can play it


# How to control the spaceship

LEFT: Go left

RIGHT: Go right


SPACE: Shoots a bullet

UP: Pause the game
